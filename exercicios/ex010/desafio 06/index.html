Tags em HTML

Em HTML5, podemos inserir os elementos da lista a seguir. Passa o mouse
sobre os termos para descobrir qual tag deverá utilizar em cada caso.

  • Parágrafos - Use a tag < p >
  • Títulos - Use a tag < h1, h2, ... h6 >
  • Imagens - Use a tag < img >
  • Abreviações - Use a tag < ins >
  • Códigos-fonte - Use a tag < shift + ! + enter >
  • Listas numeradas - Use a tag < ol - li >
  • Listas com desmacadores - Use a tag < ul - li >
