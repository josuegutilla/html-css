Principais Formatações

Negrito / Destaque

Nessa frase temos um termo em negrito usando a tag B (não sematica).

Nesta frase, tenho um termo em Destaque usando a tag STRONG (semântica).

Itálico / Ênfase

Nesta frase, temos um termo em itálico usando a tag I (não semântica).

Nesta frase, temos um termo em ênfase usando a tag EM (semântica).

Podemos também cria um texto marcado usando a tag MARK.

Texto Grande e Pequeno

Estamos criando um texto grande e um texto pequeno nesse mesmo parágrafo.
(letra grande, TAG = BIG) e (letra pequena, TAG = SMALL)

Texto Deletado

podemos marcar um texto como excluído para indicar que ele precisa ser lido, mas não considerado.
(TAG = del).

Texto Inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois.
(Tag = ins)

Texto Sobrescrito

Para Inserir coisas do tipo x2+3. TAG = SUP

Para Inserir coisas do tipo H2O. TAG = SUB
